⚪ Painel de Dados Educacionais Municipais

Sistema web desenvolvido para organizar, visualizar e facilitar o acesso a dados das escolas do município.

⚪ Objetivo

Centralizar informações educacionais em uma interface simples e intuitiva, permitindo consulta rápida de número de vagas, 
quantidades de alunos e turno das aulas, facilitando o acesso dos pais na hora da matrícula.

⚪ Funcionalidades

Visualização organizada das escolas

Exibição de informações de matrículas.

Caixa de mensagens para tirar dúvidas dos usuários. 

Interface responsiva

Estrutura preparada para integração com planilhas ou banco de dados

⚪ Problema que Resolve

Reduz o deslocamento dos pais ou outros interessados nas informações das matriculas das escolas, podendo ter o acesso a 
informações pelo celular, notebooks, tabletes ou pc.

⚪ Tecnologias Utilizadas

HTML

CSS

JavaScript

nodejs

Render Dashboard

Api Telegram
